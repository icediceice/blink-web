App Vault — Advanced Setup

This one-time setup gives Blink the ability to completely remove apps from existence in SOS mode — not just cover them, but hide them from the launcher, app list, and Settings entirely. No trace left behind.

Video walkthrough — watch before you start · approx. 8 min

Requires Permission Setup first Accounts temporarily removed Data unaffected
Accounts must be temporarily removed. Android requires all accounts (Google, Samsung, Microsoft, etc.) to be signed out before this level of system access can be granted. This page detects your accounts and guides you through each removal — you re-add them immediately when done. Your apps and data are not affected.
Before you start: You need Developer Options and USB Debugging enabled. If you haven't done the standard permission setup yet, complete that first — it takes 3 minutes.

Requirements: Chrome or Edge on desktop, USB cable, USB Debugging already on.
